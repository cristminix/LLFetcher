      - [Instructor] Whether we realize it or not, machine learning is all around us. Have you ever wondered, how does a machine learn? What about AI, deep learning, statistical modeling? Are these all different names for the same thing? Hi, I'm Fred Nwanganga. I'm a data scientist, teacher, and author with a passion for AI and machine learning. Welcome to my LinkedIn Learning course on Machine Learning with Python. In this course, I introduce what it means for machines to learn and the different ways in which machines learn. I walk through the steps of data collection, exploration and preparation with guided examples of how to accomplish each step using Python. Finally, we bring it all together as we build, evaluate, and interpret the results of a machine learning model in Python. Let's get started.
